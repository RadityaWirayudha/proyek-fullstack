# PENGERJAAN.md: Skill Fit Test - Full Stack Programmer

## 1. Ringkasan Tugas
Tugas ini bertujuan untuk membangun aplikasi administrasi RT untuk mengelola iuran bulanan, pengeluaran operasional, serta data penghuni dan rumah. Aplikasi ini ditujukan untuk mempermudah pengelolaan data warga di sebuah perumahan elite.

## 2. Kriteria Teknis (Wajib Dipatuhi)
*   **Arsitektur:** Backend dan Frontend harus dibuat secara terpisah (Decoupled)[cite: 1].
*   **Tech Stack:**
    *   **Backend:** Framework PHP – Laravel[cite: 1].
    *   **Frontend:** Library Javascript – React[cite: 1].
    *   **Database:** DBMS – MySQL[cite: 1].
*   **Constraint Utama:** **DILARANG menggunakan Docker** sebagai basic pengerjaan[cite: 1].

## 3. Fitur Utama (Requirements)
Aplikasi harus memiliki fungsi-fungsi berikut[cite: 1]:

### A. Pengelolaan Penghuni
*   **Aksi:** Tambah/Ubah data penghuni.
*   **Atribut:** Nama Lengkap, Foto KTP, Status Penghuni (Kontrak/Tetap), Nomor Telepon, Status Pernikahan (Sudah/Belum)[cite: 1].

### B. Pengelolaan Rumah
*   **Aksi:** Tambah/Ubah rumah, Tambah/Ubah penghuni rumah.
*   **Fitur:**
    *   Status rumah: Dihuni / Tidak dihuni.
    *   Histori penghuni: Catatan sejarah siapa saja yang pernah menempati rumah tersebut.
    *   Histori pembayaran: Menampilkan status pembayaran (Lunas/Belum) per rumah[cite: 1].

### C. Pengelolaan Pembayaran
*   **Iuran:** Mencakup Iuran Kebersihan (15k) & Iuran Satpam (100k).
*   **Fleksibilitas:** Mendukung pembayaran bulanan atau tahunan (khusus iuran kebersihan).
*   **Report:** 
    *   Grafik summary (Pemasukan vs Pengeluaran) selama 1 tahun.
    *   Detail report pengeluaran & pemasukan untuk bulan tertentu[cite: 1].

## 4. Output yang Harus Dihasilkan
Pastikan semua poin berikut tersedia saat penyerahan tugas[cite: 1]:
1.  **ERD (Entity Relationship Diagram):** Dokumentasi struktur database.
2.  **Repo Aplikasi:** *Source code* yang sudah diunggah ke Git.
